Livepatch is active by default on any newly-attached Ubuntu 14.04 ESM or 16.04 ESM system

Ubuntu 14.04 ESM 16.04 ESM, 18.04 LTS and 20.04 LTS
To activate Livepatch ua enable livepatch
To deactivate ua disable livepatch
Other Ubuntu versions

Livepatch is available only on LTS versions. Consider upgrading to Ubuntu 20.04 LTS.
